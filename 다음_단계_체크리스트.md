### 3단계: 백엔드 엔드포인트 구현 (필수)
현재 앱은 다음 엔드포인트를 호출합니다:
- `POST /api/auth/passkey/login`

**백엔드에서 구현해야 할 내용:**
1. Passkey credential 검증
2. 사용자 인증 처리
3. JWT/Firebase 토큰 발급
4. 로그인 성공 응답

**요청 형식:**
```json
{
  "credential": {
    "id": "...",
    "rawId": "...",
    "response": {
      "authenticatorData": "...",
      "clientDataJSON": "...",
      "signature": "...",
      "userHandle": "..."
    },
    "type": "public-key"
  }
}
```

## 🐛 문제 발생 시

### 빌드 오류가 나는 경우
1. `.\gradlew clean` 실행
2. Android Studio에서 "File" > "Invalidate Caches / Restart"
3. 다시 빌드

### Passkey 로그인이 안 되는 경우
1. **Google Play Services 최신 버전 확인**
2. **기기에 생체인증 설정 확인** (지문/얼굴 인증)
3. **로그 확인**: `adb logcat | grep PasskeyService`

### "등록된 Passkey가 없습니다" 오류
- 정상입니다! Passkey 등록 기능을 먼저 구현해야 합니다.
