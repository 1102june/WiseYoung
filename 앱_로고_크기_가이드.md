# 앱 로고 크기 가이드

## 📱 현재 로고 사용 위치

### 1. 앱 아이콘 (Launcher Icon)
- 위치: `mipmap-*` 폴더들
- 용도: 홈 화면 아이콘, 앱 목록 아이콘
- 파일: `wy_logo.webp`, `wy_logo_foreground.webp`, `wy_logo_round.webp`

### 2. 화면 내 로고 (In-App Logo)
- 위치: `drawable/wy_logo.png`
- 용도: Splash, Welcome, Auth, Login, Complete, Home 화면에서 표시
- 현재 사용 크기: 100dp, 128dp, 160dp

---

## 🎨 권장 로고 크기

### 1. 앱 아이콘 (Launcher Icon) - 필수

Android는 다양한 화면 밀도에 맞춰 여러 크기의 아이콘을 제공해야 합니다:

| 폴더 | 해상도 | 권장 크기 (픽셀) | 실제 크기 |
|------|--------|-----------------|-----------|
| `mipmap-mdpi` | 160 dpi | **48 x 48 px** | 0.75x |
| `mipmap-hdpi` | 240 dpi | **72 x 72 px** | 1.0x |
| `mipmap-xhdpi` | 320 dpi | **96 x 96 px** | 1.5x |
| `mipmap-xxhdpi` | 480 dpi | **144 x 144 px** | 2.0x |
| `mipmap-xxxhdpi` | 640 dpi | **192 x 192 px** | 3.0x |

**권장 작업 크기:**
- **원본 이미지: 1024 x 1024 px** (또는 512 x 512 px)
- 투명 배경 PNG 또는 SVG
- Android Studio의 Image Asset Studio로 자동 생성 가능

### 2. 화면 내 로고 (In-App Logo) - 권장

화면에서 사용되는 로고는 더 큰 크기로 준비하는 것이 좋습니다:

| 용도 | 권장 크기 | 형식 |
|------|----------|------|
| **기본 로고** | **512 x 512 px** 또는 **1024 x 1024 px** | PNG (투명 배경) |
| 고해상도 대응 | 1024 x 1024 px 이상 | PNG (투명 배경) |

**현재 사용 크기:**
- SplashActivity: 160dp
- WelcomeActivity: 128dp
- AuthActivity: 100dp
- HomeScreen: 다양한 크기

**권장:**
- **512 x 512 px** (최소)
- **1024 x 1024 px** (권장, 고해상도 대응)
- 투명 배경 PNG

## 📐 디자인 가이드라인

### 앱 아이콘 디자인 원칙
1. **안전 영역 (Safe Zone)**
   - 아이콘의 중요한 부분은 중앙 80% 영역에 배치
   - 가장자리 10%는 잘릴 수 있음

2. **형태**
   - 정사각형 또는 원형
   - Android는 둥근 모서리 자동 적용

3. **스타일**
   - 단순하고 명확한 디자인
   - 작은 크기에서도 인식 가능해야 함
   - 배경색과 대비가 명확해야 함

### 화면 내 로고 디자인 원칙
1. **크기**
   - 최소 512 x 512 px
   - 고해상도 화면 대응을 위해 1024 x 1024 px 권장

2. **형식**
   - PNG (투명 배경)
   - 또는 SVG (벡터 이미지)

3. **용도별 크기**
   - 큰 화면 (Splash): 160dp = 약 512px (xhdpi 기준)
   - 중간 화면 (Welcome): 128dp = 약 410px
   - 작은 화면 (Auth): 100dp = 약 320px

## 🛠️ 작업 방법

### 방법 1: Android Studio Image Asset Studio 사용 (권장)

1. **Android Studio에서:**
   - `app` → 우클릭 → `New` → `Image Asset`
   - `Launcher Icons (Adaptive and Legacy)` 선택
   - 원본 이미지 업로드 (1024 x 1024 px 권장)
   - 자동으로 모든 해상도 생성

### 방법 2: 수동 생성

1. **원본 이미지 준비:**
   - 1024 x 1024 px PNG (투명 배경)
   - 또는 SVG (벡터 이미지)

2. **리사이즈 도구 사용:**
   - 온라인: [Android Asset Studio](https://romannurik.github.io/AndroidAssetStudio/icons-launcher.html)
   - 또는 Photoshop, GIMP 등 이미지 편집 프로그램

3. **파일 배치:**
   - 앱 아이콘: 각 `mipmap-*` 폴더에 배치
   - 화면 로고: `drawable/wy_logo.png`에 배치

## 💡 추가 팁

1. **벡터 이미지 사용:**
   - SVG를 사용하면 모든 크기에 대응 가능
   - Android Vector Drawable (XML)로 변환 가능

2. **테스트:**
   - 다양한 기기에서 테스트
   - 작은 화면에서도 선명하게 보이는지 확인

3. **파일 크기:**
   - 앱 아이콘: 각 파일이 너무 크지 않도록 (최적화)
   - 화면 로고: 품질과 크기의 균형

## 📝 요약

**가장 중요한 크기:**
- **앱 아이콘 원본: 1024 x 1024 px**
- **화면 로고: 512 x 512 px (최소) 또는 1024 x 1024 px (권장)**

이 크기로 준비하면 Android Studio의 Image Asset Studio를 사용하여 자동으로 모든 해상도를 생성할 수 있습니다.
