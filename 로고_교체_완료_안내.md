# 로고 교체 완료 안내

## ✅ 완료된 작업

1. **화면 내 로고 교체 완료**
   - 파일: `app/src/main/res/drawable/wy_logo.png`
   - 용도: Splash, Welcome, Auth, Login, Complete, Home 화면에서 표시

## 📱 다음 단계: 앱 아이콘 교체 (선택사항)

앱 아이콘(홈 화면 아이콘)도 교체하려면 Android Studio의 Image Asset Studio를 사용하세요:

### 방법 1: Android Studio Image Asset Studio 사용 (권장)

1. **Android Studio에서:**
   - `app` 폴더 → 우클릭
   - `New` → `Image Asset` 선택
   - `Launcher Icons (Adaptive and Legacy)` 선택
   - `Path`에서 `C:\Users\USER\Downloads\wiseyoung_final_logo.png` 선택
   - `Next` → `Finish`
   - 자동으로 모든 해상도(mipmap-*) 생성됨

### 방법 2: 수동 교체 (고급)

각 mipmap 폴더에 적절한 크기로 리사이즈한 이미지를 배치:
- mipmap-mdpi: 48 x 48 px
- mipmap-hdpi: 72 x 72 px
- mipmap-xhdpi: 96 x 96 px
- mipmap-xxhdpi: 144 x 144 px
- mipmap-xxxhdpi: 192 x 192 px
