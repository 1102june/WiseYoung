## 📱 앱 아이콘 교체 방법

### 방법 1: Android Studio Image Asset Studio 사용 (가장 쉬움) ⭐ 권장

1. **Android Studio 열기**
2. **프로젝트 뷰에서:**
   - `app` 폴더 → 우클릭
   - `New` → `Image Asset` 선택
3. **설정:**
   - `Icon Type`: `Launcher Icons (Adaptive and Legacy)` 선택
   - `Path`: `C:\Users\USER\Downloads\wiseyoung_final_logo.png` 선택
   - `Name`: `wy_logo` (기존 이름 유지)
   - `Shape`: 원하는 모양 선택 (None, Circle, Square, Rounded Square)
   - `Background Layer`: 배경색 설정 (현재는 검은색)
   - `Foreground Layer`: 로고 이미지 자동 적용
4. **Next** → **Finish** 클릭
5. 자동으로 모든 해상도 생성됨!

### 방법 2: 수동 교체 (고급 사용자용)

원본 이미지를 각 해상도에 맞게 리사이즈하여 배치:

```
app/src/main/res/
├── mipmap-mdpi/
│   └── wy_logo.webp (48 x 48 px)
├── mipmap-hdpi/
│   └── wy_logo.webp (72 x 72 px)
├── mipmap-xhdpi/
│   └── wy_logo.webp (96 x 96 px)
├── mipmap-xxhdpi/
│   └── wy_logo.webp (144 x 144 px)
└── mipmap-xxxhdpi/
    └── wy_logo.webp (192 x 192 px)
```

**온라인 도구 사용:**
- [Android Asset Studio](https://romannurik.github.io/AndroidAssetStudio/icons-launcher.html)
- 원본 이미지 업로드 → 다운로드 → 각 폴더에 배치

## ⚠️ 주의사항

1. **투명 배경 필요 시:**
   - 현재 로고는 검은 배경입니다
